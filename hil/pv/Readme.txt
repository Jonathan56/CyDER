The FMU is generated from the model in modelica\CyDER.mo.
The Modelica model is CyDER.PvModel.Models.PvOrientated_FMU.
The model will be exported with Dymola 2018 with
  translateModelFMU("CyDER.PvModel.Models.PvOrientated_FMU", false, "pv", "2", "all", false);
Please make sure that Dymola2018 is setup to export String parameters.
